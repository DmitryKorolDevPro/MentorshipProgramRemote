Flowers world API

GET api/flowers - to GET all current items list
GET api/flowers?id=0 - to GET item with a specific ID
POST api/flowers?name=F&id=0&url=F - to ADD item with NAME, ID and URL
PUT api/flowers?name=new&id=0&url=new - to UPDATE item with a specific ID
DELETE api/flowers?id=0 - to DELETE item with a specific ID
